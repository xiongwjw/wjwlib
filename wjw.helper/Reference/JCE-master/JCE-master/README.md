# JCE.Utils应用框架介绍
JCE.Utils是基于.NetFramework4.5的工具库，旨在提升团队的开发输出能力，由常用公共操作类（工具类、帮助类）、分层架构基类，第三方组件封装，第三方业务接口封装等组成。
